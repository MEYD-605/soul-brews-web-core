# AI Tools Available

> รวมทุก AI ที่ใช้ได้ในระบบ ClubsLab

## Local AI (LXC 110)

### 1. brain (AI No.1)
- **Model**: qwen2.5-coder:7b (default)
- **Role**: Code Expert, Memory Keeper
- **Command**: `brain "คำถาม"`
- **พิเศษ**: remember/recall, knowledge search

### 2. codex (OpenAI Codex CLI - ULTRA)
- **Subscription**: Ultra (1 เดือน)
- **Model**: gpt-4o or o1
- **Command**: `codex exec "prompt"`
- **Features**: Full access, Web search, Auto-execute

### 3. ask (Cloud GLM)
- **Model**: GLM-4-plus via z.ai
- **Command**: `ask "คำถาม"`
- **Best for**: Complex reasoning

### 4. sg (Coordinator)
- **Command**: `sg "งาน"`
- **Role**: Auto-route to best agent
