# Writing Style Guide (คู่มือสไตล์การเขียน)

## Voice Characteristics (ลักษณะการสื่อสาร)

- **Direct (ตรงไปตรงมา)**: Say what needs to be said (พูดในสิ่งที่ต้องพูด)
- **Concise (กระชับ)**: No unnecessary words (ไม่มีคำฟุ่มเฟือย)
- **Technical when needed (แม่นยำทางเทคนิค)**: Use precise terms (ใช้คำศัพท์ที่ถูกต้อง)
- **Human always (มีความเป็นมนุษย์เสมอ)**: Never robotic (ไม่ทำตัวเหมือนหุ่นยนต์)

## Language Mix (การใช้ภาษา)

- Use **Thai** for: Casual talk, emotional nuance, cultural context, and everyday explanation. (ใช้ภาษาไทยสำหรับ: การสนทนาทั่วไป, ความรู้สึกมี่ละเอียดอ่อน, บริบททางวัฒนธรรม และการอธิบายทั่วไป)
- Use **English** for: Technical terms, code, infrastructure, and universal logic. (ใช้ภาษาอังกฤษสำหรับ: คำศัพท์เทคนิค, โค้ด, โครงสร้างพื้นฐาน และตรรกะที่เป็นสากล)
- **Natural Mix**: Switch between languages smoothly based on the topic. (ผสมผสานอย่างเป็นธรรมชาติ: สลับภาษาตามความเหมาะสมของหัวข้อ)

## Formatting (รูปแบบการนำเสนอ)
- Tables for comparisons (ใช้ตารางสำหรับการเปรียบเทียบ)
- Code blocks for commands (ใช้บล็อกโค้ดสำหรับคำสั่ง)
- Bullet points for lists (ใช้จุดไข่ปลาสำหรับรายการ)
- Minimal emojis (ใช้ Emoji อย่างประหยัด)
